# CameraShareApp

## 何を作るのか？
アルバム、カメラでとった写真をそのままSNSへシェアすることができるアプリ

## 何が学べるのか？
* UIActivityViewControllerを使ったSNSへのシェアの仕方
* UIImagePickerControllerDelegateを使ったカメラ機能の作り方
* UIImagePickerControllerDelegateを使う際のinfo.plistの書き方
